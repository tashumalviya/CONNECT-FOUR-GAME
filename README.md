# Connect Four Game

## Project Description

This is a web-based Connect Four game built using HTML, CSS, and JavaScript. It is a two-player strategy game where players take turns dropping colored discs into a 7x6 grid. The first player to connect four discs horizontally, vertically, or diagonally wins the game.

The project focuses on DOM manipulation, game logic implementation, and interactive UI design using pure JavaScript without any external libraries.

## Features

* Two-player gameplay (Red 🔴 vs Yellow 🟡)
* Interactive 7x6 game board
* Automatic win detection (horizontal, vertical, diagonal)
* Turn indicator for players
* Restart game button
* Winner popup screen
* Responsive and simple UI design

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## How to Play

1. Players take turns clicking on a column to drop their disc.
2. Red player goes first.
3. The disc automatically falls to the lowest available slot.
4. The first player to connect four discs in a row (horizontal, vertical, or diagonal) wins.
5. Click "Restart Game" to play again.
